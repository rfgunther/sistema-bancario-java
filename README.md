# Sistema Bancário em Java (Terminal)

Projeto prático para solidificar **Programação Orientada a Objetos (POO)** em Java puro (sem frameworks).

## Objetivo
Criar um sistema bancário simples via terminal com:
- Criação de clientes e contas
- Depósito, saque e transferência
- Extrato e persistência em arquivo

## Tecnologias
- Java 17
- Compilação manual (javac/java)

## Como rodar
```bash
javac src/main/java/br/com/banco/App.java
java -cp src/main/java br.com.banco.App
